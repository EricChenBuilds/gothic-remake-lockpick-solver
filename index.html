Gothic Remake Lockpick Solver

Set up a lock, then press Solve for the shortest click sequence.

HOW IT WORKS

In the game, the metal plates are stacked together — the closest is Plate 1, and the farthest has the largest number. These plate numbers also correspond to the plate sequence on this website.

Each plate has 7 holes. Hole 1 to Hole 7 are arranged from left to right.

HOLES (1–7)

Each plate has 7 holes. Click to select the start pin position; the target position defaults to Hole 4.

Start pin — click to set the start position

Target pin — the target position (default Hole 4, no need to set)

LINK GRID

Each plate has a small grid on its right. Each column in the grid points to another plate. Click a cell to cycle through the following options:

· None — does not affect that plate

S Same — the affected plate moves in the same direction

O Opposite — the affected plate moves in the opposite direction
